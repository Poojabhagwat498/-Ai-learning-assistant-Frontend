🚀 AI Learning Assistant (MERN + WebRTC + Socket.IO)
An advanced AI-powered learning platform built using the MERN stack that enables users to collaborate, study, and interact in real-time. This project integrates WebRTC for video/audio communication and Socket.IO for real-time features, creating a seamless group study experience.

🔥 Key Features
🔐 Authentication System (JWT-based secure login/signup)

📊 Dashboard with user activity overview

👤 User Profile Management

📄 Document Upload & Management

🧠 Flashcards Generation & Learning

❓ Quiz Creation & Practice

👥 Group Study Rooms

📞 Real-Time Video Calling (WebRTC)

🔄 Live Requests & Notifications (Socket.IO)

✅ To-Do Task Management

Authentication: JWT

⚙️ Installation & Setup

1️⃣ Clone the repository
git clone https://github.com/your-username/ai-learning-assistant.git
cd ai-learning-assistant
2️⃣ Install dependencies
For backend
cd server
npm install
For frontend
cd client
npm install
🔑 Environment Variables
Create a .env file in the server folder:

PORT=8000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
CLIENT_URL=http://localhost:5173
▶️ Run the Project
Start Backend
cd server
npm run dev
Start Frontend
cd client
npm run dev
